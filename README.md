# 💻 Portfólio Pessoal - Pedro Marin

Este é o meu portfólio pessoal desenvolvido para centralizar meus projetos, habilidades e trajetória acadêmica. O foco principal foi criar uma interface limpa, moderna e totalmente responsiva.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais da Web, priorizando performance e boas práticas de codificação:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização avançada com Flexbox para layouts dinâmicos.
* **DevIcon:** Biblioteca de ícones para representação visual da stack tecnológica
 ## 🎨 Funcionalidades

- [x] **Layout Responsivo:** Adaptado para dispositivos móveis, tablets e desktops.
- [x] **Tech Stack Visual:** Seção com ícones das tecnologias que domino (JS, Java, SQL, etc).
- [x] **Seção de Projetos:** Links diretos para meus trabalhos realizados e hospedados no GitHub Pages.
- [x] **Efeito Hover:** Interatividade nos ícones e links para uma melhor experiência do usuário (UX).
